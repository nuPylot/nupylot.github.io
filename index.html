Full-Stack AI for Robotics

  • DORA-Moveit: An open-source robotics project that integrates the DORA dataflow architecture with motion planning for robotic arms. It demonstrates a complete robotic arm control system capable of both simulation (via MuJoCo) and real hardware control, with collision-free motion planning and workflow automation for tasks like multi-viewpoint capture. The project combines inverse kinematics, motion planning (e.g., RRT-Connect), trajectory execution, and real robot interfaces into a dataflow-driven pipeline for flexible research and development.
  • DORA-Nav: A robotics navigation project built around the DORA framework, focusing on modular and dataflow-driven robot navigation. It provides navigation capabilities such as path planning, motion control, and environment interaction, organized as interconnected processing nodes within DORA, enabling flexible composition, easy experimentation, and clear separation between perception, planning, and control.
  • DORA-Viz: A modular plugin-based 3D visualization framework for robotics applications, featuring displays for sensor data (point clouds, images, laser scans), TF transforms, interactive markers, and robot models. The system allows extending displays, panels, tools, and view controllers.
